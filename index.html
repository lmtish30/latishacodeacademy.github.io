CSS Color Specificity

Color Rule Description Example
Named Colors basic name colors - up to 140 options blue, red, green
Hexadecimal Number systems with 16 digits, 0-9 followed by "A-F"; always begins with "#" #23F41A or #2FA instead of #22FFAA
RGB rgb() syntax with one value for red, one value for green, and the last value for blue; values range from 0-255 rgb(7, 210, 50)
HSL stands for hue (the color itself), saturation (the intensity of the color), and lightness (how light or dark a color is); hue ranges from 0 to 360 and saturation and lightness are both represented as percentages like this: hsl(200, 20%, 50%) hsl(200, 20%, 50%)
You can add opacity to color in RGB and HSL by adding a fourth value, a, which is represented as a percentage.
